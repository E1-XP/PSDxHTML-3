## Installation

How to run project locally:

1. Clone repository:
```bash
  git clone git@github.com:E1-XP/PSDxHTML-3.git
```
2. Move inside project directory:
```bash
  cd PSDxHTML-3
```
3. Install dependencies:
```bash
  npm install
```
4. Run development server:
```bash
  npm run start
```
